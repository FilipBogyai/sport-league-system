What this is all about

This project came to life as a result of home assignment in university course focused on advanced J2EE and JBoss technologies. The goal of this project is to integrate various Red Hat technologies in order to get a simple Enterprise Java application. It is open-source project, available on GitHub. Some ot the technologies are mentionned below:

  • JBoss Application Server 7.1.1 - WildFly
  • EJB
  • CDI
  • Seam
  • OpenShift
  • RestEasy
  • Arquillian
  • and other ...
